# Button

<Props of={Button} />

A button component that can be used to trigger an action.

## Simple Button
A simple button consist of a Lebel and a click handler
<Playground>
    <Button label={`Click me`} onClick={()=> console.log('Hi')}></Button>
</Playground>

## Themed button
A button can be themed with the `type` prop.
